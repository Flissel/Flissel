# Felix Baumann

**Founder & Technical Lead, VibeMind**

---

VibeMind is an AI-augmented desktop assistant platform built on a neuroscience-inspired multi-agent architecture. It combines 43 specialized cognitive modules with 14 AutoGen agents across five interconnected spaces (Brain, Coding, Design, Video, Ideas) to deliver private, on-device intelligence for knowledge workers and enterprises.

### Key Highlights

| | |
|---|---|
| **Architecture** | 43 neuroscience modules, 14 AutoGen agents, 5 cognitive spaces |
| **Privacy** | On-device inference via Ollama + optional cloud (OpenAI) |
| **Mission** | Ethical AI -- planned 501(c)(3) non-profit governance |
| **Market** | DACH enterprises, knowledge workers, privacy-conscious organizations |
| **Stage** | Pre-seed, building in public |
