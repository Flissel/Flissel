# Felix Baumann

**Founder & Technical Lead, VibeMind**

---

VibeMind is a **voice-controlled AI workspace** where natural speech routes through a swarm of 11 domain agents via an LLM-based intent classifier and enrichment pipeline. The Electron app renders a 3D bubble multiverse (Three.js) while Python orchestrates all backend intelligence.

### Key Highlights

| | |
|---|---|
| **Interface** | Voice-first via OpenAI Realtime API (Rachel agent) + text dashboard |
| **Architecture** | 11 domain spaces, 8+ backend agents, MinibookHub enrichment pipeline |
| **UI** | Electron + Three.js 3D multiverse with interactive bubbles |
| **LLM Routing** | OpenRouter auto-router with free-tier fallbacks |
| **Privacy** | On-device SQLite (21 tables), local-first with optional cloud |
| **Mission** | Ethical AI -- planned 501(c)(3) non-profit governance |
| **Stage** | Pre-seed, building in public |

### Domain Spaces

| Space | What it does |
|-------|-------------|
| **Ideas** | Bubble-based idea capture, linking, scoring |
| **Brain** | Neuroscience-inspired cognitive system (Tahlamus) |
| **Coding** | Society of Mind code generation (37+ agents) |
| **Desktop** | OS automation, screenshots, app control |
| **Research** | ZeroClaw deep web research |
| **N8n** | Workflow automation via n8n API |
| **AgentFarm** | AutoGen multi-agent team orchestration |
| **Video** | AI video production pipeline |
| **Flowzen** | Circadian intelligence & adaptive scheduling |
| **Rowboat** | Knowledge graph publishing |
| **Minibook** | Central message bus & collaboration |
