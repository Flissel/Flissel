# Felix Baumann

**Founder & Technical Lead, VibeMind**

---

VibeMind is a **voice-controlled AI workspace** where natural speech routes through a swarm of 11 domain agents via an LLM-based intent classifier and enrichment pipeline. The Electron app renders a 3D bubble multiverse (Three.js) while Python orchestrates all backend intelligence.

### Key Highlights

| | |
|---|---|
| **Interface** | Voice-first via OpenAI Realtime API (Rachel agent) + text dashboard |
| **Architecture** | 11 domain spaces, 8+ backend agents, MinibookHub enrichment pipeline |
| **UI** | Electron + Three.js 3D multiverse with interactive bubbles |
| **LLM Routing** | OpenRouter auto-router with free-tier fallbacks |
| **Privacy** | On-device SQLite (21 tables), local-first with optional cloud |
| **Scale** | ~70k LOC (Python + TS), 200+ dependencies, 18 MCP servers, 3 Electron apps, 4 DB engines, 6+ LLM providers |
| **Mission** | Ethical AI -- planned 501(c)(3) non-profit governance |
| **Stage** | Pre-seed, building in public |

### Custom Neural Research

The Brain space (Tahlamus) contains custom-trained neural architectures -- not wrappers, but original PyTorch research:

| Architecture | What it does |
|---|---|
| **TransformerCTM** | 4 custom-trained models (Spatial, Logic, Temporal, Value) with learned HaltPredictor + LoRA fine-tuning on Qwen2.5-0.5B |
| **ATM-R** | Adaptive Thalamic Multimodal Router -- differentiable routing gates with backprop through 10 modalities |
| **Hebbian Plasticity** | Online correlation-based learning for RadialAttentionNetwork (<1ms per update, gradient-free) |
| **5-Ring Radial Attention** | Sensory (64D) → Pattern (128D) → Semantic (256D) → Abstract (256D) → Meta (128D) |
| **Mamba SSM** | State Space Model integration with ATM-R routing (CUDA) |
| **43 Neuroscience Modules** | PFC, ACC, OFC, Amygdala, VTA, LC, Raphe, Claustrum, Hippocampus, Cerebellum and more |
